RFID Mapping Setup

To map RFID tags to students, add them to your Excel file:

  1. Open students.xlsx
  2. Add a column named "RFID Tag"
  3. Enter RFID tags for each student (e.g., A1B2C3D4)
  4. Save the file
  5. Restart the backend server
